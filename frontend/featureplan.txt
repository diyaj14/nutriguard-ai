# Tech Stack for Adding 3D

### React / Next.js (best)

Your site is Vercel → likely React.

Use:

```
three.js
react-three-fiber
@react-three/drei
```

Example component:

```javascript
import { Canvas } from "@react-three/fiber";
import { OrbitControls } from "@react-three/drei";

function FoodModel() {
  return (
    <Canvas>
      <ambientLight />
      <mesh>
        <sphereGeometry args={[1,32,32]} />
        <meshStandardMaterial color="red" />
      </mesh>
      <OrbitControls />
    </Canvas>
  );
}
```
